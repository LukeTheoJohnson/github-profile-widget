Adaptive light / dark theming

The card reads the viewer's GitHub theme and repaints itself — no more dark slab on a light profile.

EFFORTLow RISKLow DATANone (pure render)

GitHub renders READMEs in both light and dark mode, but the widget is hard-locked to Tokyo-Night dark. Roughly half of visitors see a heavy dark rectangle floating on a white page — the one element that doesn't belong. A @media (prefers-color-scheme) block inside the SVG swaps a palette of CSS custom properties, so the same file blends into either theme with zero extra requests.

What it needs: No new data. Refactor the ~12 hard-coded hex values into CSS variables + one light override block. ~1 evening.

Tech-stack identity ring

A small language donut in the dead space top-right — an instant read on what you build with.

EFFORTMedium RISKLow DATArepo languages (partly already fetched)

What it needs: One extra API pass over repo languages (or reuse repo.language already fetched). Add donut geometry + legend. ~1 day.

52-week contribution wall

The iconic GitHub year-grid as a full-width band — proof of consistent shipping, not a single snapshot.

EFFORTMedium RISKMedium DATAcontributions calendar (GraphQL)

The per-project 14-day strips show recent bursts but not the habit. A 52×7 contribution wall (the universally-recognised GitHub grid) says “this person ships every week” faster than any stat. It slots into the quiet band above the footer, turning dead space into the card's strongest at-a-glance credibility signal.

What it needs: Aggregate daily commit counts across repos for 365 days (GraphQL contributions API is one call). Render 364 cells. ~1–2 days.

“Currently shipping” spotlight

Promote the single hottest repo this week into a live hero card — latest commit message and a pulsing “shipping now”.

EFFORTMedium RISKLow DATAlatest commit of hottest repo

The widget is refreshed daily but feels static. Elevating the most-active repo into a distinct hero row — accent border, its latest commit message, a pulsing live dot — makes the card feel alive: “here's what I'm building right now.” It rewards return visitors with something that visibly moves and answers the first question anyone asks a builder: what are you working on today?

What it needs: One extra call for the top repo's latest commit message. Pick max-heat project, render the hero variant. ~1 day.
